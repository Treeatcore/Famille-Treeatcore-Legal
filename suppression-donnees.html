1. Qui est concerné ?

Cette page explique comment supprimer les données associées à l'application Famille Treeatcore.

Contact officiel : contact.treeatcore@gmail.com Réponse sous délai raisonnable

À savoir : l'application ne propose pas de création de compte et ne collecte aucune donnée personnelle sur un serveur. Toutes les informations liées à votre expérience (prénoms favoris, listes de suivi, préférences) sont stockées uniquement sur votre appareil et peuvent être supprimées directement par vous (voir section 3).

2. Procédure pour demander la suppression

2.1 Demande par e-mail

Si vous souhaitez une assistance pour la suppression de vos données, ou si vous avez contacté le support et souhaitez que vos échanges soient supprimés, envoyez un e-mail à : contact.treeatcore@gmail.com.

2.2 Informations à fournir

  • Objet conseillé : Suppression des données — Famille Treeatcore
  • Votre adresse e-mail de contact
  • Une description de votre demande (suppression des échanges support / assistance)

2.3 Vérification minimale

Afin d'éviter toute suppression non autorisée, nous pourrons demander une confirmation depuis la même adresse e-mail que celle utilisée pour la demande.

3. Suppression des données stockées sur l'appareil

3.1 Désinstallation

La méthode la plus simple pour supprimer toutes les données stockées localement (prénoms favoris, listes de suivi, préférences, compteurs d'utilisation) est de désinstaller l'application. Toutes les données locales sont définitivement supprimées.

3.2 Suppression via les paramètres Android

Vous pouvez supprimer les données sans désinstaller l'application :

  1. Ouvrez Paramètres sur votre appareil
  2. Allez dans Applications → Famille Treeatcore
  3. Appuyez sur Stockage
  4. Appuyez sur Effacer les données (et/ou Vider le cache)
Note : cette opération réinitialisera l'application à son état initial (prénoms favoris, listes cochées, préférences — tout sera remis à zéro).

3.3 Gestion du consentement publicitaire

Si l'application propose un bouton « Consentement publicitaire » (dans À propos), vous pouvez y modifier vos choix. Vous pouvez aussi réinitialiser l'identifiant publicitaire dans les réglages Android (Paramètres → Google → Annonces, selon la version et le fabricant).
